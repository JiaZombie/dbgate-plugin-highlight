# dbgate-plugin-highlight

Use DbGate for install of this plugin
- *vscode* 暗色风格
- 选中项目增加更易察觉的背景色
- 删除线改成更明显的红色
- 当前选中标签页增加明显颜色的下边框
- 选中输入框增加明显边框颜色
- 鼠标 *hover* 行或选中行整行高亮
